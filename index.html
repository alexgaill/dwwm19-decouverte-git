Commandes git

Git init

On initialise le dépôt (repository) grâce à la commande git init

git config

git config permet de définir la configuration de git sur votre machine. Cette commande n'est à utiliser que lorsque vous souhaitez la modifier et non à chaque projet

On peut par exemple la première fois faire git config --global (on modifie la configuration au global sur notre machine) user.name "mon nom"

git config --global user.email "mon@email.fr" pour changer l'email de l'auteur

git add

Permet d'ajouter les nouveaux fichiers (untracked) et les fichiers ayant des changements à la liste des fichiers à ajouter à la nouvelle version

La commande utilisée est git add [nom_du_fichier]

Une fois utilisée, les fichiers deviennent added

git add . est la commande générique qui permet d'ajouter tous les fichiers

git commit

Une fois les fichiers ajoutés on utilise git commit pour valider la version à laquelle on ajoute une description

La commande est git commit -m "Description"
